# Odyssey Grid

Trying to implement Odyssey grid in Clean Architecture.

## Installation

You need to have [Node.js 20+](https://nodejs.org/) installed on your machine.

Then run :

```bash
git clone git@github.com:cmnemoi/odyssey-grid.git
cd odyssey-grid
yarn install
yarn dev
```
